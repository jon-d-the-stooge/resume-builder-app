# Resume Content Ingestion

Desktop application for parsing and managing resume content in an Obsidian vault.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Testing

```bash
npm test
```

## Build

```bash
npm run build
```

## Project Structure

- `src/main/` - Electron main process
- `src/renderer/` - Electron renderer process (UI)
- `src/components/` - Business logic components
- `src/types/` - TypeScript type definitions
- `src/tests/` - Unit and property-based tests
